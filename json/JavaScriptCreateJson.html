在 JavaScript中创建JSON 对象

JSON:JavaScript对象表示法（JavaScript Object Notation。 JSON是存储和交换文本信息的语法。类似XML。 JSON比XML更小、更快，更易解析。 JSON使用JavaScript 语法来描述数据对象，但是 JSON 仍然独立于语言和平台。 对于AJAX 应用程序来说，JSON比XML更快更易使用. JSON 数据的书写格式是：名称/值对。名称/值对写一个冒号，然后是值。多个名称、值对用逗号分隔开。 如JSON语句{ "firstName":"Mike" ,"lastName":"John" }等价于Javascript语句firstName="Mike",lastName="John"
